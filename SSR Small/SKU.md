# SKU — IoTextra SSR Small

```
040203vvvlcc
```

| Field | Description |
|---|---|
| `040203` | Product code — IoTextra SSR Small |
| `vvv` | Version (e.g. 302 = version 3.02) |
| `l` | Load voltage / Load current |
| `cc` | Communication |

**Load voltage / Load current (l):**

| Value | Description |
|---|---|
| 1 | 350 VAC / 0.12 A |
| 2 | 60 VDC / 0.5 A |
| 3 | 40 VDC / 2 A |
| 4 | 30 VDC / 4 A |

**Communication (cc):**

| Value | Description |
|---|---|
| 10 | HOST-10 connector |
| 09 | HOST-12 connector |
| 11 | HOST-12 connector and two I²C connectors |

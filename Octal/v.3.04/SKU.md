# SKU — IoTextra Octal

```
040302vvvrcc
```

| Field | Description |
|---|---|
| `040302` | Product code — IoTextra Octal |
| `vvv` | Version (e.g. 304 = version 3.04) |
| `r` | Relay model |
| `cc` | Communication |

**Relay model (r):**

| Value | Description |
|---|---|
| 1 | HF49FD |
| 2 | APAN3105 |

**Communication (cc):**

| Value | Description |
|---|---|
| 10 | HOST-10 connector |
| 11 | HOST-12 connector and two I²C connectors |

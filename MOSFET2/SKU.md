# SKU — IoTextra MOSFET2

```
040206vvvmpc
```

| Field | Description |
|---|---|
| `040206` | Product code — IoTextra MOSFET2 |
| `vvv` | Version (e.g. 302 = version 3.02) |
| `m` | MOSFET variant (load voltage / load current) |
| `p` | DC-DC converter |
| `c` | Communication |

**MOSFET variant (m):**

| Value | Description |
|---|---|
| 1 | 30 VDC / 9 A |
| 2 | 60 VDC / 7 A |
| 3 | 90 VDC / 6 A |

**DC-DC converter (p):**

| Value | Description |
|---|---|
| 1 | 5V→5V |
| 2 | 5V→5V tall |
| 3 | 2× (5V→5V tall) |
| 4 | 5V→12V tall |

**Communication (c):**

| Value | Description |
|---|---|
| A | HOST-10 connector |
| 9 | HOST-12 connector |
| B | HOST-12 connector and two I²C connectors |

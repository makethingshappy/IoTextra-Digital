# SKU — IoTextra Octal2

```
040303vvvscc
```

| Field | Description |
|---|---|
| `040303` | Product code — IoTextra Octal2 |
| `vvv` | Version (e.g. 302 = version 3.02) |
| `s` | Signal type |
| `cc` | Communication |

**Signal type (s):**

| Value | Description |
|---|---|
| 2 | Bipolar |
| 3 | Extended |

**Communication (cc):**

| Value | Description |
|---|---|
| 10 | HOST-10 connector |
| 09 | HOST-12 connector |
| 11 | HOST-12 connector and two I²C connectors |
